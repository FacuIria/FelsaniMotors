Felsani Motors API

Ejemplos disponibles:

  • Gestión de Fotos Ejemplo de subida, visualización y eliminación de fotos utilizando multipart/form-data y Base64

Endpoints API REST disponibles:

GET /api/publicaciones - Listar todas las publicaciones GET /api/publicaciones/{id} - Obtener una publicación por ID POST /api/publicaciones - Crear una nueva publicación GET /api/publicaciones/{id}/fotos - Obtener fotos de una publicación POST /api/publicaciones/{id}/fotos - Subir una foto a una publicación
